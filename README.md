## Instalasi
1. Clone repository ini:
   ```bash
   git clone https://github.com/username/tugas-akhir-flutter.git
   ```

2. Pastikan Anda sudah menginstall Flutter di perangkat Anda.

3. Jalankan perintah berikut untuk mengunduh dependencies:
   ```bash
   flutter pub get
   ```

4. Jalankan aplikasi:
   ```bash
   flutter run
   ```
